# Quick Wins

The highest-return prompts across the journey — organized by the moment you're in. Copy, paste, fill in the `[brackets]`, send.

---

## Meetings

**After any meeting** — tracked follow-ups in 5 min

```
Summarize the meeting "[meeting name]" from [today/this morning]. Give me:
- Decisions made
- Action items, each with an owner and a due date
- Any open questions that were not resolved
Format the action items as a table I can paste into an email.
```

**Before a 1:1** — prep in 2 min

```
Help me prep for my 1:1 with [name]. Summarize what we've been working on together
over the last two weeks across email and Teams, flag anything still open or waiting
on either of us, and suggest 3 topics worth raising. Keep it to half a screen.
```

**Before any meeting** — agenda from context

```
I'm running a [30-minute project sync] with [audience] about [topic].
Based on [these notes / this background], draft an agenda with time slots
and a one-line goal for each item. Include a parking lot bucket at the end.
```

## Email & Communication

**Long email thread** — catch up instantly

```
Summarize this email thread: what's the core issue, what options have been discussed,
where do things stand now, and what (if anything) is being asked of me?
```

**Tough email** — rewrite for audience

```
Rewrite this email to be about 40% shorter and more direct, with a polite,
confident tone for a skeptical executive who skims. Lead with the ask,
keep one clear next step, and cut anything defensive. Don't change the facts.
```

**Exec briefing** — from background materials

```
Based on [this document / the attached materials], draft a 5-point exec briefing
for a 10-minute conversation with [audience]. Lead with the business outcome.
Keep each point to one sentence. Flag the one thing most likely to generate a question.
```

## Planning & Strategy

**Weekly status** — from your activity

```
Based on my last 5 working days of email, Teams messages, and meetings, draft a
5-bullet status update for my manager — what shipped, what's in progress, and
what's blocked. Keep each bullet to one line, plain and skimmable.
```

**Start of week** — plan from calendar

```
Look at my calendar and inbox for this week. Tell me:
1. My three most important commitments and what I need to be prepared for each
2. Any open threads or pending decisions I should close before Friday
3. One meeting or task I can safely deprioritize this week
```

**New project** — first-draft plan

```
Draft a first-cut project plan for [goal] by [deadline]. Break it into phases with
milestones, suggest an owner role for each workstream, and list the top 5 risks with
a mitigation for each. Put the milestones in a table. Flag any assumptions you made.
```

**Stuck on a problem** — structured options

```
I'm trying to [problem statement].
Generate 5 approaches. For each: one-sentence description, key benefit,
biggest risk, and what it would take to execute.
Push me toward the one you'd recommend and include one I probably haven't considered.
```

## Research & Documents

**Long document** — role-specific summary

```
Summarize this document for a [your role]. Give me the 3 things I most need to
know, anything that affects [your key concern], and any risks or open questions
I should raise. Skip the background — assume I know the basics.
```

**Adapt for a different audience**

```
Rewrite this [document / email / summary] for [target audience].
They care most about [outcome]. Adjust tone to be [more direct / less technical].
Remove or explain jargon. Make the key ask unmissable — put it first.
Target length: [3 bullets / one page / 200 words].
```

## Agents & Delegation (Stage 2+)

**Find answers in your org** (BizChat)

```
Find information about [topic] from across our company's SharePoint, emails,
and Teams. Summarize what you find and tell me where each piece came from.
```

**Auto-recap a meeting** (Facilitator agent)

```
[During or after a Teams meeting]
@Facilitator summarize this meeting: decisions, action items with owners,
and open questions. Send the recap to all attendees.
```

**Delegate a multi-step task** (Cowork)

```
[Research / synthesize / draft] [deliverable] using [these source documents / this topic].
The output should be [format] for [audience]. Flag any gaps where you need more information.
```
