# Stage 4 · Agent Builder

> **Build a simple agent — no code. Turn the task you keep delegating into a reusable, shareable agent right inside Copilot.**

**The shift:** from "I delegate this task each time" to "I built an agent that does it." This is your first time on the *building* side of the ramp — declarative, no-code, and low-stakes. It's the bridge between using Copilot and engineering with it.

---

## When you're ready for this stage

You've handed Copilot the same kind of multi-step task more than once and thought *"I wish this just existed."* You're ready to point an agent at your own files or a SharePoint site and give it a persona — without writing code.

## What you need

- Access to the agent-building surface in Copilot (Agent Builder / Copilot Studio's lightweight authoring).
- A clear, narrow job for the agent and a knowledge source (files, a SharePoint site).
- A small group willing to pilot it and give feedback.
