# Start by Stage

> **Find your rung on the ladder in two minutes, then take the next concrete step.**

This site is a ladder, not a library. You don't need to read it front to back — find where you are now and take the next step.

!!! tip "Prefer to start from your job, not your skill level?"
    Head to **[Start by Role](start-by-role.md)** — the same walkthroughs, grouped by how you actually spend your day.

---

## The 2-minute self-check

Read each rung in order. The **last statement you can say "yes, comfortably" to** is roughly where you are today — start there.

| Rung | You can comfortably say… | If that's your ceiling, start here |
|---|---|---|
| **1 · Chat** | "I open Copilot in Teams, Word, or Outlook several times a week and get useful results." | [Stage 1 · Chat](stages/stage-1-chat.md) → [Turn a meeting into follow-ups](walkthroughs/chat-meeting-followups.md) |
| **2 · First-Party Agents** | "I've used one of the included agents — Researcher, Analyst, or Facilitator — on real work." | [Stage 2](stages/stage-2-first-party.md) → [the agents in your license](walkthroughs/first-party-agents-overview.md) |
| **3 · Cowork** | "I've handed Copilot a multi-step task and let it work through it — not just one prompt." | [Stage 3 · Cowork](stages/stage-3-cowork.md) → [hand off an end-to-end task](walkthroughs/cowork-end-to-end-task.md) |
| **4 · Agent Builder** | "I've built (or could build) a simple no-code agent over my own files or a SharePoint site." | [Stage 4](stages/stage-4-agent-builder.md) → [a team-knowledge agent](walkthroughs/agent-builder-team-knowledge.md) |
| **5 · Copilot Studio** | "I've published an agent with a knowledge source and at least one real action." | [Stage 5 · Studio](stages/stage-5-studio.md) → [your first Studio agent](walkthroughs/studio-first-agent.md) |

!!! note "You don't have to climb one rung at a time"
    Jump straight to where you are — but each stage is designed to make the next feel obvious rather than a cliff. The shift at each rung:

    **Chat** — use it in the flow of work → **First-Party** — use what's already built for you → **Cowork** — let it run multi-step work → **Agent Builder** — build something simple, no code → **Studio** — build for production.

## If you only do one thing today

Open **[Turn a meeting into tracked follow-ups](walkthroughs/chat-meeting-followups.md)**, copy the prompt, and run it against your most recent meeting. It's the fastest way to feel what the ramp is about — and everything else builds from there.
