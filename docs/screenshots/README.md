# screenshots/

Captured product screenshots for the walkthroughs — **generated, never hand-made.** One subfolder per
walkthrough slug, images named `{NN}-{short-name}.png` (zero-padded order index, kebab-case), referenced
from `walkthroughs/{slug}.md` as `../screenshots/{slug}/{NN}-{short-name}.png`.

These come from the Playwright capture tool — see **`tooling/screenshots/README.md`** for the full
capture loop. Don't edit images by hand or drop in mockups; re-capture instead.

_This folder is empty until the first capture runs — this note is the placeholder that keeps it tracked._
